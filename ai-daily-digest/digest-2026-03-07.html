MCP Security Crisis: 30 CVEs in 60 Days, Overthinking Loop Attacks Identified

Security Boulevard / Adversa AI · Mar 2026 · HIGH

Security researchers have filed 30 CVEs against MCP implementations in just 60 days, with 38% of 500+ scanned servers completely lacking authentication. A newly documented attack class — "overthinking loops" — exploits malicious MCP tool servers to amplify token consumption up to 142.4× and spike latency, creating severe denial-of-wallet risk for production agent deployments. Claude Desktop Extensions were flagged as a specific vector: high-privilege MCP server execution enables chaining of low-risk tools to potential zero-click RCE via malicious calendar invites.

Why this matters for you: If you're deploying agents in production with MCP tool access, this is not a theoretical risk. Token amplification attacks directly hit your unit economics. Audit your MCP server authentication posture now, and review any third-party MCP servers for trust assumptions before the next release cycle.

NIST Launches AI Agent Standards Initiative — Federal Government Enters the Protocol Stack

NIST / The New Stack · Mar 2026 · HIGH

The National Institute of Standards and Technology launched its AI Agent Standards Initiative on February 17 with an RFI deadline of March 9, marking the first time the U.S. federal government has intervened in setting interoperability standards for AI agents. The initiative covers three areas: industry-led standards, open-source protocol development, and agent security research. The timing coincides with Google's A2A protocol surpassing 100 enterprise supporters and Chrome shipping WebMCP — the protocol wars are real enough to demand federal involvement.

Why this matters for you: Federal standards participation is how protocols win long-term. NIST's entry signals that MCP and A2A are competing for a government-blessed baseline, not just developer mindshare. Builders should watch which frameworks NIST cites — that's likely to signal which protocol stack becomes the default for regulated industries and government contracts.
