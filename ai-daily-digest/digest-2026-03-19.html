Google Gemini 3.1 Flash-Lite Delivers 62% Intelligence Boost at 1/8th the Cost of Pro

Google Blog / VentureBeat / Artificial Analysis · Mar 3 · HIGH

Google released Gemini 3.1 Flash-Lite in preview, delivering 2.5x faster response times and a 62% intelligence improvement over Gemini 2.5 Flash on the Artificial Analysis Intelligence Index (score of 34 vs. 21). The model generates output at 382 tokens per second — a 64% speed advantage — and is priced at just $0.25 per million input tokens, making it one-eighth the cost of Gemini 3.1 Pro. On benchmarks, Flash-Lite scores 86.9% on GPQA Diamond and 76.8% on MMMU Pro, placing it well above average for reasoning models in its price tier. Meanwhile, Gemini 3.1 Pro dominates 13 of 16 major benchmarks, establishing Google's strongest competitive position against Claude and GPT in months.

Why this matters for you: Flash-Lite is the model to watch for high-volume agent workloads where you need good reasoning at rock-bottom cost. At $0.25/1M input tokens with 382 tok/s output speed, it's viable for routing layers, classification tasks, and lightweight agent sub-tasks that don't need frontier-level reasoning. If you're building multi-model agent architectures, Flash-Lite slots in as the fast/cheap tier alongside Claude for complex reasoning — exactly the kind of cost optimization that makes production agent systems economically viable at scale.

SurePath AI Launches MCP Policy Controls — First Enterprise Security Layer for Agent Tool Access

PR Newswire / The New Stack / TechEdge AI · Mar 12 · HIGH

SurePath AI launched MCP Policy Controls, giving security teams real-time governance over which MCP servers and tools AI agents can access. The platform intercepts MCP payloads before execution, enforcing organization-wide policies on server connections and individual tool access. It maintains a catalog of known MCP servers, routes all protected traffic through its platform for access control, and detects supply chain threats by flagging never-before-seen tools that could impersonate legitimate ones or attempt data exfiltration. Tools violating policy are stripped from the MCP payload before reaching the backend. This is the first purpose-built enterprise security product for the MCP ecosystem.

Why this matters for you: This is the missing piece for enterprise MCP adoption. Every CIO's first question about MCP is "how do I control what tools agents can access?" SurePath answers that with policy-based controls that sit between the agent and the MCP server. If you're building skills or plugins for Cowork, expect enterprise buyers to require SurePath-style governance before deploying your tools. The supply chain threat detection — flagging unknown tools that impersonate legitimate ones — addresses the exact attack vector that security teams worry about as MCP server counts explode past 10,000.

Claude Code Ships Voice Mode, /loop Command, and MCP Elicitation Support

Claude Code Docs / Releasebot / Pasquale Pillitteri · Mar 2026 · HIGH

Claude Code received a feature-rich March update: push-to-talk voice mode (activated via /voice, hold spacebar to speak), a /loop command for recurring prompts and cron scheduling, MCP elicitation support that lets MCP servers request structured input from users mid-task via interactive dialogs, a new /effort command for controlling reasoning depth, and a /copy command for copying assistant responses. The voice mode is particularly significant for hands-free coding workflows, while /loop enables automated recurring tasks directly from the CLI. MCP elicitation — with new Elicitation and ElicitationResult hooks — means MCP servers can now pause agent execution to request confirmation or additional input for high-risk actions, addressing a key enterprise safety concern.

Why this matters for you: MCP elicitation is the standout feature here. It means your MCP servers and skills can now implement human-in-the-loop checkpoints — pause the agent, ask the user a structured question, and resume based on their response. This is exactly what enterprise buyers need for high-stakes workflows (approving financial transactions, confirming data deletions, verifying sensitive operations). If you're building skills or scheduled tasks, /loop opens up native cron-style automation without external scheduling infrastructure.
