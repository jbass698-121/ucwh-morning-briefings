Model & Platform Updates

OpenAI Launches GPT-5.4 — Native Computer Use, 1M-Token Context, Agentic Workflows

OpenAI / TechCrunch · Mar 5 · HIGH

GPT-5.4 is OpenAI's most capable model to date, and the first general-purpose model with native computer-use capabilities — agents can now operate desktop applications, navigate browsers, and execute multi-step workflows across software. Available in three variants: standard, Thinking (extended reasoning for Plus/Pro users), and Pro (enterprise API). The 1M-token context window is the largest OpenAI has offered. On OSWorld-Verified, GPT-5.4 achieves 75% success rate, surpassing human performance at 72.4%. Hallucination rates are down 27% when using web tools and 20% on internal knowledge vs. GPT-5.2.

Why this matters for you: Native computer-use changes the agent game. Instead of building tool integrations one-by-one, agents can now interact with any desktop application the way a human would. Combined with the 1M context window, this opens up workflows that were previously impossible — like having an agent process an entire quarter's worth of reports in a single pass. Worth testing against your existing Claude Code and Cowork workflows to benchmark cost-per-task.

Why this matters for you: MIT-licensed with SWE-bench scores rivaling closed models means GLM-5 is immediately useful for on-premise or cost-sensitive deployments. The Huawei Ascend training story is the bigger signal — if Chinese labs can match frontier performance on domestic silicon, the global chip supply chain dynamics shift permanently.

Platinum members include Anthropic, OpenAI, AWS, Google, Microsoft, Cloudflare, Block, Bloomberg, Intuit, Replit, Samsung, PayPal, Salesforce, and SAP. MCP's Python and TypeScript SDKs now exceed 97 million monthly downloads with 10,000+ published servers. This formalizes MCP as the de facto standard for agent-to-tool communication under neutral governance.

Why this matters for you: MCP under the Linux Foundation is the clearest buy signal yet for building on the protocol. Every major cloud provider and AI lab is now committed to interoperability through AAIF. If you're building agent infrastructure, MCP is no longer a bet — it's the standard. The three-layer architecture (MCP for tools, A2A for agent-to-agent, WebMCP for web) is the stack to build against.

MCP Security Under Siege: 30 CVEs in 60 Days, Overthinking Loop Attack Discovered

Adversa AI / Security Research · Mar 2026 · HIGH

MCP gateways and dedicated AI security tools are emerging as essential enterprise infrastructure layers.
